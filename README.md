# PheWAS Analysis (All of Us)

This repository contains phenotype-wide association studies (PheWAS) conducted using the All of Us Research Program dataset.

## Analyses
- LDLR variant PheWAS
- Smoking status PheWAS
- Polygenic risk score (PRS) PheWAS
- PRS comparison analysis

## Methods
- Logistic regression
- Covariates: age, sex, EHR length, ancestry PCs
- Phecodes derived from ICD codes
- Multiple testing correction using Bonferroni threshold

## Outputs
- PheWAS results tables (.tsv)
- Manhattan plots (.png)

## Note
Due to All of Us data use policies, individual-level data is not included.
